# love.touch

<b><i>
Provides an interface to touch-screen presses.
</b></i>

| Name                                                          | Description                                            | Notes                         |
|---------------------------------------------------------------|--------------------------------------------------------|-------------------------------|
| [getTouches](https://love2d.org/wiki/love.touch.getTouches)   | Gets a list of all active touch-presses                |                               |
| [getPosition](https://love2d.org/wiki/love.touch.getPosition) | Gets the current position of the specified touch-press |                               |
| [getPressure](https://love2d.org/wiki/love.touch.getPressure) | Gets the current pressure of the specified touch-press | This always returns one (1.0) |
